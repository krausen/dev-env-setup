# dev-env-setup
Set up my development environment with this Ansible playbook

## Prerequisites
  - Python >= 2
  - MacOS >= 10.15
  - Signed in to Mac App Store

## How to?
Install ansible

```shell
pip install ansible
```

Run Ansible playbook (Make sure that you are signed in to Mac App Store)

```zsh
ansible-playbook dev-env.yml
```
